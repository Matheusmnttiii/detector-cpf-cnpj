# Detector de CPF/CNPJ

Ferramenta web para formatação e validação de CPF e CNPJ em tempo real, com feedback visual ao usuário.

## 🔗 Demo
https://matheusmnttiii.github.io/detector-cpf-cnpj/

## 💻 Tecnologias
- HTML
- CSS
- JavaScript

## ✨ Funcionalidades
- Validação automática de CPF e CNPJ
- Formatação dinâmica durante a digitação
- Feedback visual de erro ou sucesso
- Interface simples e responsiva

## 📚 Aprendizados
Implementação de algoritmos de validação, manipulação de eventos de input e criação de feedback imediato ao usuário.
